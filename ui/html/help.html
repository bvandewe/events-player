Help and Usage

This micro-app serves as a test subscriber for CloudEvents. It exposes a POST /events/pub endpoint that Publishers can use to send events. Upon reception of a valid CloudEvent, the app distributes it to all browsers that are connected at that time, if any. It can also generate test events and send them to the selected CloudEvent Gateway. (Gateways can be added by Administrators with an Environment variable).

Open the generator, select the localhost Gateway, and send an event. It will appear in the UI.
Select another Gateway, send an event, and if the event matches a subscription directed towards the CloudEvent Player, it should receive the event back.

Connection Status

  • The connection to the backend is established and refreshed every 15s.
  • An event was received within the last 10s.
  • The connection to the backend is broken. Trying to re-establish it every 2s.

Keyboard Shortcuts

  • CTRL or META+UpArrowOpen the Generator
  • Escape or META+UpDownClose the Generator
  • META+fFocus/Unfocus on the Search/Filter box
